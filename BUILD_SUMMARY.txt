🎯 API ENDPOINTS IMPLEMENTED

User Discovery:
  GET  /api/users/available
       - Fetch available users for pairing
       - Pagination support (offset, limit)
       - Returns user list with pairing status

Trading System:
  POST /api/pairs/[id]/trade
       - Create trade (mint, buy, sell, list)
       - Validates trade parameters
       - Updates price history and volume

Like System:
  GET  /api/stories/[id]/like
       - Fetch like count for a story
  POST /api/stories/[id]/like
       - Like a story with wallet validation

Comments:
  GET  /api/stories/[id]/comment
       - Fetch comments with pagination
  POST /api/stories/[id]/comment
       - Create comment with validation (max 500 chars)

Send Value (USDC):
  GET  /api/stories/[id]/value
       - Fetch transfer history with stats
  POST /api/stories/[id]/value
       - Send USDC (min 0.7) with verification hooks
